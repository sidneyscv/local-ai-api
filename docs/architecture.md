# Architecture

## Overview

LOCAL-AI-API is a modular AI platform designed for:

- AI agents
- Local LLM orchestration
- Workspace management
- API automation
- Multi-provider AI integrations

## Main Components

### apps/api
Backend services and REST APIs.

### apps/web
Frontend web interface.

### apps/desktop
Desktop application.

### packages/ai-core
AI orchestration and providers.

### packages/database
Database abstraction layer.

### packages/shared
Shared utilities and types.

### packages/ui
Reusable UI components.
